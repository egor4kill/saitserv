Сюда клади скриншоты для галереи.

Чтобы галерея не жрала трафик, нужны ДВЕ версии каждого фото:
1. thumb  — превью для сетки, ширина ~500px, .webp, обычно 30-60 КБ
2. full   — версия побольше для лайтбокса (открывается по клику), ширина ~1600px, .webp, обычно 150-300 КБ

Полный размер грузится ТОЛЬКО когда кто-то кликает по фото, а не всем подряд при
открытии страницы — поэтому вес full не так критичен, как вес thumb.

Как сделать на squoosh.app:
1. Кидаешь фото → выбираешь WebP → ширина 500 → сохраняешь как name-thumb.webp
2. То же фото → ширина 1600 → сохраняешь как name-full.webp

Вписываешь оба пути в config/gallery.json.
